# Moodle-PM - Gestão de Projetos na Educação

Este é um projeto acadêmico de exemplo estilo Moodle focado em **Gestão de Projetos na Educação**, integrando recursos avançados de **Gamificação** (sistema de pontuação XP e leaderboard/ranking de alunos) e um **Chatbot Acadêmico** para respostas de dúvidas conceituais.

Acesse as instruções completas de instalação, execução e detalhes sobre a arquitetura no arquivo de documentação complementar:
👉 [Documentação Detalhada em docs/README.md](docs/README.md)

## 🚀 Como Executar Rapidamente

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicialize o servidor:
   ```bash
   npm start
   ```
3. Abra a interface no navegador em: [http://localhost:3000](http://localhost:3000)
